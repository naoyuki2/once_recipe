# アプリ名

「今日の一皿」

# 説明

- アクセスするとランダムで楽天レシピAPIから取得したレシピが表示される
- 次の日になると表示されるレシピが更新される
- １日に１レシピしか表示されない
- ログインすることによってレシピ保存機能が使えるようになり、保存したレシピを見返すことができる
